# Vodaphone Technical Test FE

## Demo

https://lnesi.github.io/vdptest/index.html

### Install

```
git clone https://github.com/lnesi/vdptest.git
```

#### Requirements
Node +9
```
npm install
```

### Development
Start local devleopment server
```
npm run start
```
### Build
Build compile version for host
```
npm run build
```
### Deploy

Deploy to Github Pages

```
npm run deploy
```
### Unit Testing

```
npm run test
```
